# How to use README.md

## Header
Header, sub-header is denoted by the number of hash at the start of it

## Backticks

Backticks (``) can be used to write code 

For example, you can use the command `print("Hello World")` to print Hello World to the terminal.

## Bold text and italics

Use double asterisk for bold

For example, **This is a bold text**

For example, *This is a italic text*

## Image

You can also attach image, by attaching the source with the command 

`<img src="Attach link here" />`

For example, attached is an example of the Queen's Gambit opening in chess
